# Sustainability & Green Daily Digest and Alerts Telegram Bot

Curated daily updates, trend alerts, and personalized notifications for Sustainability & Green professionals. Stay informed without information overload.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
